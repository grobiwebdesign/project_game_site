Welcome

GamesBok is an experimental website hosting games made using modern technology, for the purpose of providing an entertaining learning experience for people on the Autistic Spectrum.

While GamesBox is still currently in development, it has been released early for testing and feedback purposes. As a result of being a University project, the games currently featured were imported from HTMLGames.com to keep to the time schedule.

The intention is to later replace these embedded games with our own authentic games made using the same technology, with help from Phaser to improve the cross-device gaming experience.
